BH1010 - PageIsCallback

Access to IsCallback property of System.Web.UI.Page

IDBH1010
Type NamePageIsCallbackAnalyzer
CategoryCmsApiReplacements
SeverityWarning
PackageBugHunter.Analyzers

Code Fixes

A code fix will be presented. The access will be replaced with RequestHelper.IsCallback() expression.

Before:

var isCallback = Request.IsCallback;

After:

using CMS.Helpers;
...
var isCallback = RequestHelper.IsCallback();

Code fix limitations

If the member access is conditional (SyntaxKind.ConditionalAccessExpression) or the invoked method has conditional access as its Expression property, no code fix will be provided. The logic would be very complicated and the fix would not have to be correct. Read more about codefixes on conditional accesses in this issue on GitHub.
