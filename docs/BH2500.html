BH2500 - ValidationHelperGet

One of methods GetDouble, GetDecimal, GetDateTime or GetDate of CMS.Helpers.ValidationHelper class is invoked from a WebPart file

IDBH2500
Type NameValidationHelperGetAnalyzer
CategoryCmsApiGuidelines
SeverityWarning
PackageBugHunter.Web.Analyzers

Remarks

Diagnostic will only be raised for a file which is located in WebParts folder. This means its relative path contains one of:

  • CMSWebParts\
  • CMSModules\AdminControls\Controls\UIControls\

Code Fixes

A code fix will be presented.

If a culture is passed as a third argument (either string or a CultureInfo instance), this argument will be stripped off as part of the code fix.

Before:

using CMS.Helpers;
...
var a = ValidationHelper.GetDouble("42", 42, "en-us");
var b = ValidationHelper.GetDateTime("", System.DateTime.MaxValue, "en-us");
var c = ValidationHelper.GetDate("", System.DateTime.MaxValue, "en-us");
var d = ValidationHelper.GetDecimal("", 0, "en-us");

After:

using CMS.Helpers;
...
var a = ValidationHelper.GetDoubleSystem("42", 42);
var b = ValidationHelper.GetDateTimeSystem("", System.DateTime.MaxValue);
var c = ValidationHelper.GetDateSystem("", System.DateTime.MaxValue);
var d = ValidationHelper.GetDecimalSystem("", 0);
